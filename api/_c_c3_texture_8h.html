Returns the CCTexturePixelFormat corresponding to the specified OpenGL pixel format and type.

Not all combinations of OpenGL pixel format and type can be mapped to a corresponding CCTexturePixelFormat value. In those cases, this function returns CCTexturePixelFormat_Default.
